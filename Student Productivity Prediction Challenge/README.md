# Student Productivity Prediction Challenge
## About the Challenge
The goal of this competition is to build machine learning models that predict student productivity_score using behavioral, academic, and lifestyle features.
The objective of this competition is to predict the **productivity_score** of students based on their behavioral and academic data.
Participants will use machine learning techniques to analyze study habits, distractions, sleep, and performance features.

## Evaluation Metric
The challenge's metric is RMSE.
